
# Welcome to Intwana Hub!

This guide will help you understand what Intwana Hub is, what you can do on the platform, and how you can contribute to its development.

## What is Intwana Hub?

Intwana Hub is a vibrant online community designed for young people to connect, learn, and play. Our mission is to provide a safe and engaging digital space where you can:

*   **Play educational games:** Challenge yourself with a variety of fun and interactive games.
*   **Compete with friends:** See how you rank on the leaderboard and compete for the top spot.
*   **Showcase your creativity:** Share your projects and creations with the community.
*   **Discover new opportunities:** Find information about scholarships, workshops, and other opportunities.

## Getting Started

To get started with Intwana Hub, you'll need to create an account. Here's how:

1.  **Sign Up:** Go to the "Sign Up" page and create a new account.
2.  **Log In:** Once you've created your account, you can log in to access all the features of the platform.
3.  **Explore:** Start exploring the different sections of the app, such as the games, leaderboard, and showcase.

## For Contributors

We welcome contributions from everyone! If you're a developer and want to help improve Intwana Hub, here's how you can get started:

### Running the Project Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/okuhlecharlieman/intwana-hub.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up Firebase:**
    *   Create a new Firebase project.
    *   Copy your Firebase configuration and add it to `src/lib/firebase.js`.
4.  **Run the development server:**
    ```bash
    npm run dev
    ```

We're excited to have you as part of our community! If you have any questions, feel free to reach out to us.
